IoT Agriculture Embedded Systems

Smart Agriculture Monitoring System

An end-to-end IoT solution for precision farming, featuring real-time soil monitoring, automated irrigation, and cloud-based analytics for smallholder farmers in Kenya.

Completed: December 2024
Duration: 4 Months
Role: Lead Hardware Engineer

Problem

Farmers losing 40% of crops due to improper irrigation timing and lack of soil data visibility.

Solution

Low-cost wireless sensor nodes transmitting soil moisture, pH, and weather data to a central dashboard.

Impact

30% reduction in water usage and 25% increase in crop yield during pilot program.

Technical Overview

Hardware Architecture

  • Microcontroller: ESP32-S3 for dual-core processing and WiFi/BT connectivity
  • Communication: LoRaWAN for long-range, low-power farm-wide coverage
  • Power: Solar panel + LiPo battery management system with sleep mode optimization
  • Sensors: Capacitive soil moisture, DHT22 (temp/humidity), pH probe, rainfall detector

Software & Firmware

  • Firmware: C++ on Arduino framework with FreeRTOS for task management
  • Protocol: MQTT over TLS for secure data transmission to AWS IoT Core
  • Backend: Node.js API with MongoDB for time-series data storage
  • Frontend: React Native mobile app for farmers with SMS fallback alerts
