Programming & Algorithms

Navigation Stack (PROS + C++) 

// Pure Pursuit Algorithm for path tracking
void followPath(vector<Point> path, double lookAhead) {
    while (currentIndex < path.size()) {
        Point target = findLookAheadPoint(path, lookAhead);
        double curvature = calculateCurvature(target);
        setDriveVelocity(calculateLeftVel(curvature), 
                        calculateRightVel(curvature));
        pros::delay(10);
    }
}

PID Control Systems

  • • Drive PID: Maintains straight-line motion
  • • Turn PID: Precise heading control (±1° accuracy)
  • • Flywheel PID: RPM consistency for accuracy
  • • Vision PID: Aiming alignment with target

Sensor Fusion

  • • Inertial Sensor: 3-axis gyro/accelerometer
  • • Optical Encoder: Wheel odometry
  • • Vision Sensor: Object detection (200Hz)
  • • Distance Sensor: Wall detection
